<h1>Glownius - SkinCare web application ✨</h1> 
A dynamic and responsive web application for exploring, purchasing, and managing skincare products.

## Features

<li>Extensive Product Catalog: </li>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;  Explore a wide range of skincare products, empowering users to find the perfect match for their unique needs.
<li>Intuitive Search and Filtering:</li> 
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;  Easily navigate the product catalog using advanced search and filtering options based on type, brand, ingredients, or specific concerns.
<li>Seamless Purchase Experience: </li>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Securely purchase products directly through the website with smooth transaction processing.
<li>Personalized Account Management: </li>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Create and manage accounts, view order history, and track personal preferences for a streamlined experience.
<li>Responsive Design: </li>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Enjoy a user-friendly interface that adapts flawlessly across various devices, ensuring optimal usability on desktops, and mobile phones.

## Technologies Used
<li>Frontend: React.js, JavaScript</li>
<li>Backend: Node.js, Express.js</li>
<li>Database: MongoDB</li>
